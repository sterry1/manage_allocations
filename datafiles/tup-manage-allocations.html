Welcome to TACC's Projects and Allocations Management System!

We were unable to locate any projects associated with your account, but as an eligible PI you are able to create any new project(s) and request any applicable allocation(s) by following these steps:
  • Fill out and submit the project creation form below. After submission, this page will show a table of information associated with your new project.

  • To add any associated grant information, click on the "Grants" row to expand its contents. Click on the "Add New Grant" button to display the new grant form for you to fill out and submit.

    Repeat the process for any number of grants at any time you wish. You may also update any grant information at any time.

  • To add any associated publication information, click on the "Publications" row to expand its contents. Click on the "Add New Publication" button to display the new publication form for you to fill out and submit.

    Repeat the process for any number of publications at any time you wish. You may also update any publication information at any time.

  • To request an allocation, click on the "Allocations" row to expand its contents. Here you will always find a summary of your current allocations and allocation requests.

    To request an allocation, click on the "Request Allocation" button to display the allocation request form. Select the resource(s) for which you would like to request an allocation, fill out any required information, and click on "Submit."

    While under review, the status of your allocation request should remain visible here. You may also update or cancel your request, as well as request allocation renewals or increases from this same allocation section.

  • To manage the users associated with your project, click on the "Users" row to expand its contents. Here you will find a list of current users under the project, including the project's PI and Delegate.

    Adding a new user is as simple as clicking on "Add New User" and searching for the user's account within the system. To remove a user from the project, click on "Remove Users" and select the users you wish to have removed. Note that this will not delete the user's account, but simply remove them from the project.

    You may also assign a user as a Delegate, or Co-PI, who would be able to manage a project/allocation on behalf of yourself. Simply click on "Assign New Delegate" and choose a new Delegate from the list of users associated with the project. Note that there can be only one Delegate at any given time.

  • To view a summary of usage information for the project, click on the "Usage" row to expand its contents. Here you will find a summary of SUs used, SUs refunded, and an overview of recent jobs run across associated resources.
Detailed help is available by clicking on the "Need help?" link that will appear after creating your first project. However, if you have any questions or run into any problems, please open a consulting ticket by clicking on the Consulting tab above.

Additionally, If you have any comments or suggestions on how we can improve project and allocation management for you, please feel free to use the portal's feedback form.

Thank you!

Welcome to TACC's Projects and Allocations Management System

TACC's project and allocations management system has been designed to provide you with a comprehensive interface from which you can view and manage your projects, allocations, and associated account details from a single location.

The page below shows a table of information concerning each of the projects your account is associated with. Clicking on each row in the project listings below allows you to expand or collapse more detailed information under each heading:
  • Project Name / Your Association: The top-most row in the listing below shows the name of each project your account is associated with, as well as the type of role your account has for that project. Your account may be listed as PI, Delegate, or Standard.

    Clicking on the Project Name row expands/collapses more detailed information associated with the project.

  • Project Information: Contains basic information describing the project, including project name, project type, field of science, and the abstract describing the project. PIs and Delegates will have the option to update the project information at any time.

  • Grants: Displays any associated grant information. For each grant a number of details may be displayed, including grant title, PI, funding agency, award number and amount, start and end dates, and NSF status code.

    PIs and Delegates have the option to add any number of grants and update grant information at any time. Currently, grant award amounts are only visible to PIs and Delegates.

  • Publications: Displays any associated publication information. Each publication associated with the project may contain details such as title, author, publication year, publisher, venue, and URLs.

    PIs and Delegates have the option to add any number of publications and update publication information at any time.

  • Allocations: Here you will find a summary of the current allocations and allocation requests associated with the project. In addition to details such as start and end dates, SUs granted, SUs remaining, and SUs refunded, the status of any allocation requests will be visible.

    PIs and Delegates have the option to request a new allocation, allocation renewal, and/or allocation increase from the same location, as well as update or cancel any pending allocation requests.

  • Users: Lists the user accounts currently associated with the project, including the account owner's name, username, email address, and role.

    PIs and Delegates are able to add current users to the project using the add new user interface to search for users by name, email, or username. To remove a user from the project, click on "Remove Users" and select the users you wish to have removed. Note this will not delete the user's account, but simply remove them from the project.

    PIs may also assign a user as a Delegate, or Co-PI, who would be able to manage a project/allocation on behalf of the PI. Simply click on "Assign New Delegate" and choose a new Delegate from the list of users associated with the project. Note there can be only one Delegate at any given time.

  • Usage: Here you will find a summary of total SUs used, total SUs refunded, and an overview of recent jobs run sorted by resource.

  • Create New Project: PIs will have an additional "Create New Project" button visible at the top of the page. Clicking on the button will display a form to enter basic project information.

    After submitting the required information, the new project will immediately appear in the project table below. Expanding the "Project Information" row will display the details you just entered.

    Continue down each section, first entering any associated grants, then entering any associated publications, before continuing on to the allocation request process.

If you have any questions or run into any problems, please open a consulting ticket by clicking on the Consulting tab above.

Additionally, If you have any comments or suggestions on how we can improve project and allocation management for you, please feel free to use the portal's feedback form.

Thank you!
